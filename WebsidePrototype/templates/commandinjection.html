Anleitung

In diesem Abschnitt wird das Thema Command Injection behandelt. Dazu soll über eine Website ein Passwort, welches auf dem Server liegt, augelesen werden.

Zugriff

Zuerst muss über den Browser auf die Website zugegriffen werden: Dort kann nun navigiert werden. Auf verschiedenen Seiten können durch Command Injection Schwachstellen ausgenutzt werden, um den Inhalt auf dem Server auszulesen.
Für den Zugriff müssen Pattern verwendet werden. Diesen können zum Beispiel hier nachgelesen werden: http://www.golemtechnologies.com/articles/shell-injection

Ziel

Ziel dieser Übung ist es ein Passwort auf dem Sever auszulesen. Dieses ist in einem File versteckt was ausfindig gemacht und ausgelesen werden muss.

Ziel

Wer eine kleine Hilfe benötigt:

- Man muss in zwei Schritten und mit zwei unterschiedlichen Arten angreifen.
